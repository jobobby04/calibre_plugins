## Overview

This plugin is for reading book metadata from the clipboard.

Using a userscript to handle getting metadata in a json format from the website, we can import metadata based on what the website has.

When you select a book in Calibre, you can then click the toolbar menu to change that books metadata to what is copied tgo the clipboard.



### Tutorials

#### Adding the Plugin to Calibre
1. Download the latest Clipboard Metadata plugin zip from https://github.com/jobobby04/calibre_plugins/releases/
2. At the top of Calibre, press `Preferences`
3. Select `Plugins` under `Advanced`
4. Click `Load plugin from file` and select the downloaded zip file

#### Copy Metadata from website example
An example script for copying metadata from a website is located [here](../scripts/TOB%20Story%20Downloader%20&%20Copier.js)
